# 你画我猜复盘器

一个本地静态 HTML 复盘工具，用来模拟手机上的「你画我猜」界面和流程，并通过右侧代码编辑器预览绘图效果。

## 使用

直接用浏览器打开：

```text
index.html
```

自动运行默认示例：

```text
index.html?autorun=1
```

## 功能

- 模拟准备、倒计时、选词、作画、猜词、结算、下一轮流程。
- 使用 1080 x 2280 的手机坐标系。
- 标出建议安全画板区域：`x=170..880, y=390..1130`。
- 按实测游戏色板限制画笔和画布颜色。
- 支持 `pen`、`sw`、`bez`、`ell`、`dot`、`star` 等绘图 API。
- 支持笔画调试：检查代码生成笔画数、实际 SVG 路径数、可见笔画数、越界、同色、过快、无效坐标等问题。
